How-to guides for building and running AI coding workflows with Archon.

## Planned Guides

The following guides are being migrated from the existing documentation:

- **Authoring Workflows** — Create multi-step YAML workflows with DAG nodes
- **Authoring Commands** — Write prompt templates for workflow nodes
- **Loop Nodes** — Iterative AI execution with completion conditions
- **Approval Nodes** — Human review gates in workflow pipelines
- **Per-Node Hooks** — Attach Claude SDK hooks to individual nodes
- **Per-Node MCP Servers** — Connect external tools (GitHub, Linear, etc.) per node
- **Per-Node Skills** — Preload specialized knowledge into node agents
- **Global Workflows** — User-level workflows that apply to every project
