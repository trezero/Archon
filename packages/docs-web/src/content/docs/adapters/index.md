Archon supports multiple platform adapters. Each adapter connects Archon to a different communication channel, letting you trigger workflows and interact with AI agents from wherever you work.

## Available Adapters

| Adapter | Transport | Status |
|---------|-----------|--------|
| **Web UI** | SSE streaming | Built-in |
| **CLI** | stdout | Built-in |
| **Slack** | Socket Mode | Requires setup |
| **Telegram** | Bot API polling | Requires setup |
| **GitHub** | Webhooks | Requires setup |
| **Discord** | WebSocket | Requires setup |
